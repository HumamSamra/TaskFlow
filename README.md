# 🚀 TaskFlow

TaskFlow is a simple and lightweight task management web application that helps users organize and track their daily tasks efficiently.

---

## ✨ Features

- 📝 Create new tasks  
- 🔄 Mark tasks as **On Going** or **Finished**  
- 🔍 Real-time task search  
- 🗑️ Clear all tasks with confirmation  
- 💾 Persistent storage using **localStorage**  

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **Bootstrap 5**
- **Font Awesome**
- **SweetAlert2**

---

## ⚙️ How It Works

- Tasks are stored locally using **localStorage**
- Each task includes:
  - Unique ID
  - Task text
  - Creation date
  - Completion status
- UI updates dynamically using JavaScript and DOM manipulation
